# Pricing — VeriWorkly Portfolio

## Draft Tier

- Price: $0/month
- Limits: Private preview only, no public subdomain
- Features: Complete editor, access to all templates, draft previews, reusable content model

## Portfolio Pro Tier

- Price: $12/month
- Limits: 1 live publication
- Features: Custom VeriWorkly subdomain (e.g., yourname.veriworkly.com), direct SEO meta title and social description configuration, visitors analytics, hosted portfolio images, watermark/logo removed
